## ğŸš€ **Getting Started**

### **Prerequisites**
- Node.js 18+ 
- npm or yarn

### **Installation**
```bash
# Clone the repository
git clone https://github.com/mbadiao/portfolio.git

# Navigate to the project directory
cd portfolio

# Install dependencies
npm install

# Run the development server
npm run dev
```

### **Build for Production**
```bash
# Build the application
npm run build

# Start the production server
npm start
```
